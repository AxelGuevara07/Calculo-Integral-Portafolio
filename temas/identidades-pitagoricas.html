Sustitución trigonométrica mediante identidades pitagóricas

Este método combina la sustitución trigonométrica con las identidades pitagóricas. Después de hacer el cambio de variable, se usan las relaciones trigonométricas y un triángulo rectángulo para regresar a la variable original.

Identidades fundamentales

\[\sin^2\theta + \cos^2\theta = 1\]
\[1 + \tan^2\theta = \sec^2\theta\]
\[\sec^2\theta - 1 = \tan^2\theta\]

Idea del método

La identidad pitagórica se utiliza para simplificar la raíz que aparece en la integral. Luego, mediante la sustitución inicial, se construye un triángulo rectángulo para expresar las razones trigonométricas en función de \(x\).

Ejemplo guiado

Resolver: \(\int \frac{dx}{\sqrt{x^2+9}}\)

Como aparece \(x^2+3^2\), hacemos \(x=3\tan\theta\). Entonces \(dx=3\sec^2\theta\,d\theta\).

\[\sqrt{x^2+9}=\sqrt{9\tan^2\theta+9}=3\sec\theta\]
\[\int \frac{3\sec^2\theta}{3\sec\theta}\,d\theta=\int \sec\theta\,d\theta\]
\[=\ln|\sec\theta+\tan\theta|+C\]

Con el triángulo rectángulo se obtiene:

\[\boxed{\ln\left|x+\sqrt{x^2+9}\right|+C}\]

Punto importante

El triángulo de referencia es el puente entre la variable trigonométrica y la variable original.
